HON Wikipedia translator
========================

Translator using wikipedia for automatic semantic translations.
As long as there is an entry in Wikipedia in both languages wanted,
a translation is provided.

Build
-----

```bash
npm install
```

Test
----

```bash
npm test
```

Run
---

```bash
npm start
```
